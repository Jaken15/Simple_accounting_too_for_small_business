# Simple_accounting_too_for_small_business
A simple desktop accounting tool designed for small businesses to track their income and expenses easily. This project helps users manage their finances by recording transactions and visualizing financial d

🚀 Features

➕ Add Income – Record income transactions.

➖ Add Expenses – Record business expenses.

📅 Date Filter – Filter transactions by specific dates.

📈 Data Visualization – Display financial data using charts.

💾 Local Database – Store transactions using SQLite.

🛠 Technologies Used

Python

Tkinter – For the graphical user interface (GUI)

SQLite – For storing financial data

Matplotlib – For generating charts and data visualization

📷 Application Overview

This application allows small business owners to:

Track daily income and expenses

Filter records by date

View simple financial charts

All data is stored locally using SQLite, making the application lightweight and easy to use without needing an internet connection.
